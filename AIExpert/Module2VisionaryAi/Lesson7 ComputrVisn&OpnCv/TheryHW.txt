What is Grayscale?
Grayscale means an image that contains only shades of gray, from black to white, without any color.
Instead of color information, each pixel represents brightness (intensity).

Pixel values in Grayscale
In a grayscale image, each pixel has one value:

Value	Meaning
0	Black
128	Medium gray
255	White

| Image Type      | Channels    | Example Pixel  |
| --------------- | ----------- | -------------- |
| Color Image     | 3 (B, G, R) | [120, 60, 200] |
| Grayscale Image | 1           | 140            |

So grayscale images contain less data and are simpler.

Why Grayscale is Used
Grayscale is commonly used because:
⚡ Faster processing
📉 Less memory usage
🔍 Better for detecting edges and shapes

Applications include:
Face detection
OCR (text recognition)
Medical imaging
Security cameras
