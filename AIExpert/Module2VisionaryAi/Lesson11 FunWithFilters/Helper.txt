1️⃣ How OpenCV Handles Colors

OpenCV images are usually stored as 8-bit unsigned integers (uint8).

Each color channel (Blue, Green, Red) can only have values from 0 to 255.

| Min | Max |
| --- | --- |
| 0   | 255 |

0 → no intensity, 255 → full intensity

filtered_image[:, :, 2] = cv2.add(filtered_image[:, :, 2], 50)  --> implies increase intensity of red channel by 50
# Suppose red value = 200
cv2.add(200, 50)   # 250 ✅ not clipped
cv2.add(200, 100)  # 255 ✅ (clipped)
cv2.add(200, 5000) # 255 ✅ (still clipped)
